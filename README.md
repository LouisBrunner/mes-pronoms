# Mes Pronoms ![Build Status][ci-image]

Ce dépôt contient le code source de [https://mes-pronoms.lbrunner.net/](https://mes-pronoms.lbrunner.net/), un outil qui permet de décrire vos pronoms français de manière unifiée.
